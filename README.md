## Project Description
This project analyzes global carbon dioxide (CO₂) emissions using publicly available data from the World Bank. The dataset measures total CO₂ emissions excluding land use, land-use change, and forestry (LULUCF), reported in million tonnes of CO₂ equivalent (Mt CO₂e).

## Research Question
What are the trends and patterns of CO₂ emissions across countries, and how do they vary over time?

## Data Source and Description
This project uses data from the World Bank:

Carbon dioxide (CO₂) emissions (total), excluding LULUCF (Mt CO₂e)

This indicator measures emissions from sectors such as energy, industry, agriculture, and waste, excluding land-use and forestry-related emissions.

Data source:
https://data.worldbank.org/indicator/EN.GHG.CO2.MT.CE.AR5?locations=1W

Due to GitHub file size limitations, only cleaned datasets used in the analysis are included in the `data/` folder.
